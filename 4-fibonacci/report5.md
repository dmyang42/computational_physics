## 计算物理 实验报告4

羊达明 PB16000647 

第五题：

用16807产生器测试随机数序列中满足关系 Xn-1 < X n < Xn+1
的比重。讨论Fibonacci延迟产生器中出现这种关系的比重。

schrage方法用第四题的代码，执行后给出了该关系的比重。本题只附上schrage方法代码及可执行程序。
使用命令 ./schrage * (*代表seed)，在终端中输出该比重。

Fibonacci N = 100000, seed = 42 该关系比重为0.1650
Fibonacci N = 100000, seed = 2333 该关系比重为0.1685

Schrage N = 100000, seed = 42 该关系比重为0.1669
Schrage N = 100000, seed = 42 该关系比重为0.1685

这些比重在执行时输出到终端中。

两种方法这种有规律的关系的比重相当。
